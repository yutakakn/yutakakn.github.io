ジャンルパズルアクション
ルール 上から４つの要素からなる１つのブロックが落ちてきます。このブロックを うまく回転、左右方向へ移動させて、適切な位置へ落とします。盤の一行が そろうと、その一行が消え、段が下がります。以上、これの繰り返しです。
操作方法
  • 左へ移動　　………　左カーソル、Ｊ、４
  • 右へ移動　　………　右カーソル、Ｌ、６
  • 回転　　　　………　上カーソル、Ｋ、５、８
  • 左回転　　　………　Ｚ
  • 右回転　　　………　Ｘ
  • 落下　　　　………　下カーソル、スペース、２
  • タイトルへ　………　Ｒ（ゲームオーバー時）
