カードシャッフル

  • ゲームを起動する
説明
ジャンルテーブルゲーム
ルール ゲームを起動すると、左から順にスペード、クローバー、ダイヤ、ハートの4枚の トランプカードが配置されてます。所持金 200 GOLDを持っていて掛け金を決める と、4枚のカードが裏返しになりシャッフルされます。最初、一番左にあった スペードがどこに移動したかを当てると、お金が増え ます。掛け金は 100 GOLD単位です。
クリア条件 所持金が 10000 GOLDを超えるとゲームクリアとなります。逆に 0 GOLDになると ゲームオーバーです。
操作方法
  • 掛け金の決定　…　キー入力したあと Enter キーで決定です。
  • カードの選択　…　マウスのクリックで裏返しになったカードを選びます。
Javaバージョン JDK 1.1(VisualCafe 3.0c)
    ※注意
    IE5.0上で動作させている場合フリーズすることがあります。そのような時は JIT機能を無効にすると起きないことがあります。JIT機能は、ツール-インターネット オプション-詳細設定-「Java JITコンパイラの使用」のチェックを外すと無効にでき ます。
