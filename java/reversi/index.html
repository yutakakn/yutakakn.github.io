リバーシ

  • ゲームを起動する
説明
ジャンルテーブルパズル
ルール 白か黒の駒を盤面に置いていき、相手の駒を自分の駒ではさむことで相手の 駒を自分の駒にひっくり返すことができます。最終的に駒の数が多い方が 勝ち。
ゲームスタート 下のメニューから『コンピュータと対戦する』か『二人で対戦する』かを選択 します。また、『先手』か『後手』かも選択できます。[Start]ボタンでゲーム 開始です。
操作方法
  • クリック　　　　‥‥‥　駒を置く
  • [Start]　　　　 ‥‥‥　ゲーム開始
  • [Retry]　　　　 ‥‥‥　ギブアップ
  • [Undo]　　　　　‥‥‥　待った（一度だけ）
  • [COMP,HUMAN]　　‥‥‥　対・コンピュータ、人の選択
  • [FIRST,SECOND]　‥‥‥　先手、後手の選択
